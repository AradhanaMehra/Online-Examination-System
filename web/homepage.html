Welcome to E-Learn. An online platform to conduct examinations online. Login as teacher to create an exam and login as a student to take examinations. You can also see the examinations created and display them.

Examinations

Create Examination

In order to create examination you need to be logged in as a teacher. Then you can create questions for the subjects accordingly.

Display all Examinations

You can display all the examinations created by the teacher by clicking here.

Display all Results

You can display all the Results of the examinations taken by the students by clicking here.

Logging in

You can log in as a teacher to create examinations and you can login as a student to take those examinations.

Student

You can log in as a student and take only those examinations that have been created by the teachers

Teacher

You can log in as a teacher and create examinations which can be taken by the students.
